Flex direction

A propriedade define que direção o conteiner que empilhar os itens flex

seus valores são:

  • column: empilha ps itens flexiveis verticalmente(de cima para baixo)
  • colum-reverse: empilha os itens flexiveis verticalmente(de baixo para cima)
  • row: empilha os itens flexiveis horizontalmente (da esqueda para a direita)
  • row-reverse: empilha os itens flexiveis horizontalmente (da direita para esqueda)

flex wrap

a propriedade especifica se os itens flexiveis devem ser ou nao, de acordo com a viewport.

seus valores sao:

  • wrap: especificaque os itens flexiveis serao enrolados, se necessario
  • nowrap:especifica que os itens flexiveis snao serao enrolados (modo padrao)
  • wrap-reverse: especifica que os itens flexiveis iram embrulhar se necessario, e, ordem inversa

justify content

propriedade e usada para lainhar os itens flexiveis

seus valores sao:

  • center: os valores de alinhamento flexiveis no centro do conteiner
  • flex-start:os valores de alinhamento flexiveis no inicio do conteiner
  • flex-end: os valores de alinhamento flexiveis no final do conteiner
  • space-around: os valores de alinhamento flexiveis com espaco antes, entre e depois das linhas

align items

a propriedade e usada para alinhar os itens flexiveis

  • center:o valor alinha os itens flexiveis no meio do conteiner

align content

a propriedade e usada para alihar as linhas flexiveis

seus valores sao:

  • center: o calor mostra as linhas flexiveis no meio do conteiner
  • stretch:o valor estica as linhas flexiveis para assumir todo o espaço restante do conteiner
  • flex-start:o valor exibe as linhas flexiveis no inicio do conteiner
  • flex-end:o valor exibe as linhas flexiveis no final do conteiner
  • space-around:o valor exibe as linhas flexiveis com o espaco antes entre e depois dentro do conteiner
  • space-between:o valor exive as linhas flexiveis com o mesmo espaco entre elas no conteiner

neste exemplo usamos um conteiner com 600px de altura com o flex-wrap, propriedade definida para wrap para melhor demostrar o align-content.
